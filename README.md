Aerial (エアリアル) - 創作支援ツール
**Aerial（エアリアル）**は、小説家、脚本家、ゲームシナリオライターのためのブラウザベースの創作支援ツールです。
「キャラクターが勝手に動き出す」ような深い設定作りと、設定同士がリンクし合うWikiのような管理体験を提供します。

🚀 主な機能
👤 キャラクター設定 & 深掘り質問
単なるプロフィールの記録に留まらず、キャラクターの内面を抉り出す「深掘り質問」機能を搭載。

カスタム質問: 「大事にしているものは？」「窮地に追い込むには？」など、ジャンルに合わせた質問項目を自由に追加・編集可能。

複数キャラクター管理: 膨大な登場人物をリスト形式で整理。

🌍 世界設定の管理
物語の舞台となる地理、組織、潮流などを体系的に管理。

キャラクター設定と同様の直感的なUIで、世界の背景を緻密に構築できます。

🔗 インテリジェント・オートリンク
設定内のテキストに「キャラクター名」や「世界設定名」が含まれていると、自動的にハイパーリンクを生成。

Wikiのような操作感: リンクをクリックするだけで、即座に対象の設定ページへジャンプし、編集が可能です。

📌 アイデアボード
コルクボードを模した直感的なメモエリア。

付箋管理: プロットの断片や思いついたネタを付箋感覚でメモ。

リアルタイム保存: 入力した内容は即座にブラウザへ保持されます。

💾 ローカルストレージ保存
サーバー不要。データはすべてユーザーのブラウザ内に保存されるため、プライバシー面でも安心です。

🛠 使い方
このリポジトリをクローンするか、index.htmlをダウンロードします。

ブラウザ（Chrome, Edge, Safari等）でindex.htmlを開きます。

サイドバーから「質問設定」を行い、自分だけのキャラクターシートを作成してから執筆を開始してください。

📂 技術スタック
Frontend: HTML5, CSS3, JavaScript (Vanilla JS)

Storage: Web Storage API (LocalStorage)

Layout: Flexbox, CSS Grid
